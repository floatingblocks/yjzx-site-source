## 备选方案

鉴于上回很多人搭建不起来平台，这样可能就没法通过常规流程提交过来。因此我决定开启这个备选方案。

把 html 或者 markdown 格式的文章和用到的资源文件一起打 zip 包发到邮箱 nethermobs@foxmail.com 。

邮件正文按照如下格式填写

``` text
作者：
标题：
分类：
标签：
简介：
```

我们会进行手动同步到网站上。

当然除非万不得已我不推荐这么做。毕竟捣鼓各种软件安装也是提升自己计算机技能的方式。
